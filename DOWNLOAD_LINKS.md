## ðŸ¤– ROS (Robot Operating System)

ROS is **NOT** available on PyPI. It must be downloaded from official ROS repositories.

### ROS 1 (Noetic - Ubuntu 20.04)

**Official Website**: http://wiki.ros.org/noetic

**Download Options**:

#### Option 1: Official Debian Packages (Recommended)
```bash
# Add ROS repository
sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu $(lsb_release -sc) main" > /etc/apt/sources.list.d/ros-latest.list'

# Add keys
sudo apt install curl
curl -s https://raw.githubusercontent.com/ros/rosdistro/master/ros.asc | sudo apt-key add -

# Update and install
sudo apt update
sudo apt install ros-noetic-desktop-full
```

#### Option 2: Manual Download from Packages.ros.org

**Base URL**: http://packages.ros.org/ros/ubuntu/pool/main/r/

**Key Packages**:
- **ros-noetic-desktop-full**: http://packages.ros.org/ros/ubuntu/pool/main/r/ros-noetic-desktop-full/
- **ros-noetic-ros-base**: http://packages.ros.org/ros/ubuntu/pool/main/r/ros-noetic-ros-base/
- **ros-noetic-rospy**: http://packages.ros.org/ros/ubuntu/pool/main/r/ros-noetic-rospy/

**Download all .deb files** and install with:
```bash
sudo dpkg -i *.deb
sudo apt-get install -f  # Fix dependencies
```

### ROS 2 (Humble - Ubuntu 22.04)

**Official Website**: https://docs.ros.org/en/humble/

**Download Options**:

#### Option 1: Official Debian Packages (Recommended)
```bash
# Add ROS 2 repository
sudo apt install software-properties-common
sudo add-apt-repository universe
sudo apt update && sudo apt install curl -y

# Add ROS 2 GPG key
sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key -o /usr/share/keyrings/ros-archive-keyring.gpg

# Add repository to sources
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu $(. /etc/os-release && echo $UBUNTU_CODENAME) main" | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null

# Update and install
sudo apt update
sudo apt install ros-humble-desktop
```

#### Option 2: Manual Download from Packages.ros.org

**Base URL**: http://packages.ros.org/ros2/ubuntu/pool/main/r/

**Key Packages**:
- **ros-humble-desktop**: http://packages.ros.org/ros2/ubuntu/pool/main/r/ros-humble-desktop/
- **ros-humble-ros-base**: http://packages.ros.org/ros2/ubuntu/pool/main/r/ros-humble-ros-base/
- **ros-humble-rclpy**: http://packages.ros.org/ros2/ubuntu/pool/main/r/ros-humble-rclpy/

## ðŸ“¥ How to Download Manually

### Method 1: Using wget (Command Line)

```bash
# Create packages directory
mkdir -p packages

# Download a package (example: matrix-nio)
wget -P packages/ https://files.pythonhosted.org/packages/.../matrix_nio-0.25.2-py3-none-any.whl

# Download all packages from a list
while read package; do
    wget -P packages/ "https://pypi.org/simple/$package/"
done < requirements-offline.txt
```

---

### Method 2: Using Browser (Manual)

1. **Go to PyPI**:
   - Visit https://pypi.org/project/PACKAGE_NAME/
   
2. **Click "Download files"**:
   - Look for the `.whl` file (wheel format)
   - Choose the one matching your platform:
     - `py3-none-any.whl` - Pure Python (works on all platforms)
     - `cp311-cp311-manylinux_2_17_x86_64.whl` - Python 3.11, Linux x86_64
     - `cp311-cp311-win_amd64.whl` - Python 3.11, Windows 64-bit
   
3. **Download**:
   - Right-click the file â†’ "Save Link As..."
   - Save to `ribit.2.0/packages/` folder

---

### Method 3: Using pip download (Easiest)

```bash
# Download a single package
pip3 download -d packages/ matrix-nio[e2e]

# Download all packages from requirements
pip3 download -d packages/ -r requirements-offline.txt
```

## ðŸ“‚ How to Copy Packages to Folder

### From Windows to Linux (USB Drive)

```bash
# On Windows (download packages)
1. Download all .whl files to a folder
2. Copy folder to USB drive

# On Linux (copy from USB)
mkdir -p ~/ribit.2.0/packages

```

---

### From Another Linux System (SCP)

```bash

cd ribit.2.0
./download_dependencies.sh

# Copy to target system
scp -r packages/ user@target-system:~/ribit.2.0/
```

---

### From Network Share

## ðŸ“ Notes

- **Platform**: Download packages for your target platform (x86_64, arm64, etc.)
- **Python Version**: Match Python version (3.11, 3.12, etc.)
- **Wheel vs Source**: Prefer `.whl` files over `.tar.gz` (faster installation)
- **ROS Size**: ROS1/ROS2 full desktop is ~2-4 GB
- **Mirrors**: Use mirrors if main PyPI is slow

## ðŸ†˜ Troubleshooting

### "Package not found"

**Solution**: Check the exact package name on PyPI:
```bash
pip3 search PACKAGE_NAME
```

### "Wrong platform"

**Solution**: Download the correct wheel for your platform:
- Linux x86_64: `manylinux_2_17_x86_64`
- Linux ARM: `manylinux_2_17_aarch64`
- Windows: `win_amd64`
- macOS: `macosx_10_9_x86_64`

### "Dependencies missing"

**Solution**: Download all dependencies recursively:
```bash
pip3 download -d packages/ --requirement requirements-offline.txt
```

This will download the package AND all its dependencies.
